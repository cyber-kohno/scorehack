Melody は、Outline で定義したコード進行と時間軸に沿って、歌メロを設計するためのモードです。

scorehack では、Melody を単なるピアノロールではなく、Chord と結びついた作曲設計の領域として扱います。  
ノートの位置、長さ、音程、発音を入力しながら、現在のコードや調性との関係を確認できます。

## Melody の責務

Melody は主に次の責務を担います。

- 歌メロのノートを入力する
- ノートの位置、長さ、音程を編集する
- ノートを分割、結合、削除する
- 複数ノートを選択してまとめて操作する
- 発音情報を入力する
- Chord に対する構成音や調性外の音を確認する
- MusicXML や MIDI に渡すメロディ情報を整える

## Outline との関係

Melody は、Outline の Chord を時間上の基準として扱います。

Chord の拍数や食い込みが変わると、Melody のカーソル位置、ノートの所属、コード構成音アシストにも影響します。  
そのため、Melody は Outline から独立した譜面ではなく、Outline の上に乗っている歌メロ編集レイヤーです。

## Arrange との関係

Arrange は、Melody を確認するための仮伴奏として機能します。

Melody で歌メロを作り、Arrange で伴奏やボイシングを付けることで、コードとメロディの関係を聴きながら確認できます。

## 基本的な作業の流れ

Melody では、概ね次の流れで作業します。

1. Outline で Chord を作る
2. Cursor の位置、長さ、音程を調整する
3. Note を入力する
4. Note を移動、伸縮、分割、結合して整える
5. Pronunciation を入力する
6. Chord Assist と Playback で確認する

最終的には、MusicXML や MIDI として外部ツールに渡せるメロディ情報を作ることを目指します。
