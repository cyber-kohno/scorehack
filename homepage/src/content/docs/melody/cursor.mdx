---
title: Cursor
description: ノート入力の基準点
---

Cursor は、次に入力する Note の基準になる情報です。

Melody では、いきなり Note を置くのではなく、まず Cursor が「どこに、どの長さで、どの高さの音を置くか」を持ちます。  
その状態で入力操作を行うと、Cursor の情報をもとに Note が作られます。

## Cursor が持つ情報

Cursor は主に次の情報を持ちます。

- 位置
- 長さ
- 音程
- norm

## 位置

位置は、タイムライン上でどこに Note を置くかを表します。

Melody の位置は、Outline の Chord に沿った時間軸上にあります。  
カーソルを左右に動かすことで、次に入力するノートの開始位置を変えます。

## 長さ

長さは、次に入力する Note の音価を表します。

たとえば、四分音符、八分音符、十六分音符のように、入力するノートの単位を切り替えながら作業します。

## 音程

音程は、次に入力する Note の高さを表します。

上下方向に Cursor を動かすことで、入力するメロディの音程を変えます。  
スケール移動やオクターブ移動など、音楽的な単位で動かす操作もあります。

## norm

norm は、Melody 内部で長さや位置を扱うための基準です。

ユーザー視点では、`1/4`, `1/8`, `1/16` のような入力単位に近い概念です。  
Cursor の norm を変えると、カーソル移動の細かさや、次に入力する Note の長さが変わります。

## 複合拍子での Cursor

6/8 などの複合拍子では、単純に八分音符を1拍として扱うのではなく、大きな拍感を前提に Cursor の入力単位が決まります。

たとえば 6/8 では、付点四分音符のまとまりを意識した入力単位が重要になります。

## Chord との関係

Cursor の位置によって、現在参照している Chord が決まります。

Chord が変わると、コード構成音アシストや調性外構成音の表示も変わります。  
そのため、Cursor は Melody と Outline をつなぐ基準点でもあります。

## Keyboard

カーソル状態では、Note をまだ選んでいない位置に対して、配置位置、長さ、音程を調整します。
