---

Selection は、Melody 上で現在操作対象になっている Note や範囲です。

scorehack はキーボード操作を中心にしているため、「今どの Note を操作しているか」を明確に持つことが重要です。

## フォーカス

1つの Note を選択している状態をフォーカスと呼びます。

フォーカスされた Note は、移動、長さ変更、分割、発音入力などの対象になります。

## 複数選択

複数の Note を選択できます。

複数選択した Note は、まとめて移動したり、条件を満たせば結合したりできます。

## 範囲選択

範囲選択は、フォーカス位置とロック位置によって表されます。

連続した Note を扱う時に使います。  
たとえば、同じ高さで隣接している複数 Note を選択して結合する、といった操作ができます。

## 複数選択中の移動

複数選択中に通常の上下移動を行うと、選択範囲の先頭または末尾へフォーカスを戻します。

これは、範囲選択を解除して次の操作に移るための動作です。

## コピーとペースト

選択中の Note はコピーできます。

コピーした Note は、Cursor の位置を基準に貼り付けます。  
貼り付け時には、スコア範囲外にはみ出さないかチェックします。

貼り付け後は、意図的にクリップボードをクリアします。

## Keyboard

複数選択中は、選択範囲全体を対象に移動、コピー、削除できます。
