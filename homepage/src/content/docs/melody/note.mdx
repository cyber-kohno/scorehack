Note は、Melody 上に実際に配置された音です。

Cursor をもとに作成され、位置、長さ、音程、発音情報を持ちます。

## Note が持つ情報

Note は主に次の情報を持ちます。

- 位置
- 長さ
- 音程
- 発音情報

## 位置

Note の位置は、タイムライン上の開始位置です。

位置は Chord の時間範囲と関係します。  
Note がどの Chord 上にあるかによって、コード構成音かどうか、調性内かどうかの意味が変わります。

## 長さ

Note の長さは、その音がどれだけ続くかを表します。

長さは Cursor の入力単位から作られますが、あとから伸縮できます。  
長さを変更する時は、隣の Note との衝突や、スコア範囲外へのはみ出しを避ける必要があります。

## 音程

Note の音程は、メロディの高さです。

音程は直接上下に移動できるほか、スケールに沿って移動する操作もあります。  
現在の調性や Chord に対して、構成音、テンション、非和声音としての意味を持ちます。

## 発音情報

Note には、発音情報を持たせることができます。

これは MusicXML エクスポートや歌声合成との連携で使います。  
scorehack では、アプリ上の入力はローマ字を中心に扱い、エクスポート時に必要な形式へ変換します。

## Note と Chord

同じ Note でも、乗っている Chord によって意味が変わります。

たとえば、あるコードでは3度の構成音でも、別のコードではテンションや非和声音になることがあります。  
Melody では、現在の Chord を見ながら Note の意味を確認できます。

## Keyboard

Note にフォーカスしている時は、選択中の Note を基準に編集します。

  </thead>
