Edit は、Melody 上の Note を作成、移動、加工する操作です。

このページでは、Melody でできる主な編集操作を整理します。

## 入力

Cursor の位置、長さ、音程をもとに Note を入力します。

入力時には、既存の Note と重ならないか、スコア範囲内に収まるかを確認します。

## 移動

フォーカスした Note は、左右に移動できます。

移動時には、次のチェックが行われます。

- スコア範囲外にはみ出さないこと
- 他の Note と衝突しないこと
- 現在の入力単位で表現できること

移動できない場合は、フォーカス位置にトーストで理由を表示します。

## 長さ変更

Note の長さを変更できます。

長さを短くする、伸ばす、どちらの場合も範囲外や他の Note との衝突を確認します。

## 分割

単一の Note を、Cursor の入力単位で分割できます。

たとえば、四分音符の Note をフォーカスし、Cursor を十六分音符単位にして分割すると、同じ高さの十六分音符が4つ並びます。

分割できる条件は次のとおりです。

- フォーカスが単一 Note であること
- Cursor の単位が Note より細かいこと
- Note の長さが Cursor の単位で割り切れること

## 結合

複数選択した Note を結合できます。

結合できる条件は次のとおりです。

- すべての Note の音程が同じであること
- すべての Note が隣接していること

条件を満たさない場合は、トーストでエラーを表示します。

## 削除

フォーカス中または選択中の Note を削除できます。

削除後は、次の操作を続けやすい位置にフォーカスを戻します。

## コピーとペースト

選択した Note をコピーし、Cursor の位置に貼り付けます。

貼り付け時には、コピー元の先頭 Note と Cursor の位置関係を基準に、複数 Note の相対位置を保って配置します。  
範囲外にはみ出す場合は貼り付けません。

## Keyboard

Edit では、Note の追加、分割、結合、削除、コピー＆ペーストを行います。

<table>
  <thead>
    <tr><th>キー</th><th>操作</th></tr>
  </thead>
  <tbody>
    <tr><td><kbd>A</kbd></td><td>カーソル位置、またはフォーカス中の Note を基準に Note を追加します。</td></tr>
    <tr><td><kbd>1</kbd> / <kbd>2</kbd> / <kbd>3</kbd></td><td>編集に使う norm を切り替えます。</td></tr>
    <tr><td><kbd>Q</kbd></td><td>単体フォーカスでは Note を分割します。複数選択では条件を満たす Note 群を結合します。</td></tr>
    <tr><td><kbd>D</kbd> + <kbd>←</kbd> / <kbd>→</kbd></td><td>Note、または選択中の Note 群を前後に移動します。</td></tr>
    <tr><td><kbd>F</kbd> + <kbd>←</kbd> / <kbd>→</kbd></td><td>単体フォーカス中の Note の長さを変更します。</td></tr>
    <tr><td><kbd>E</kbd> + <kbd>Delete</kbd></td><td>削除後に前の Note へフォーカスします。</td></tr>
    <tr><td><kbd>Delete</kbd></td><td>フォーカス中、または選択中の Note を削除します。</td></tr>
    <tr><td><kbd>Ctrl</kbd> + <kbd>C</kbd></td><td>選択中の Note をコピーします。</td></tr>
    <tr><td><kbd>Ctrl</kbd> + <kbd>V</kbd></td><td>コピーした Note を貼り付けます。</td></tr>
  </tbody>
</table>

移動、貼り付け、結合ができない場合は、フォーカス位置にエラーメッセージが表示されます。
